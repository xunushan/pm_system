# ADR-012: executor 规划态不填，按专题 type 推断

任务规划态不定 executor，pm-daily 按专题 type 运行时推断。

## Context

executor 实体有专门的规划态字段，其值始终可从专题类型推导。显式字段增加同步维护负担。

## Decision

任务规划态不定 executor，pm-daily 按专题 type 推断。推断规则：learning/research/source->人；dev/survey->智能体（规则写在 pm-daily Skill）。卡片只读展示，要改走 H5 页面。

## Consequences

- 数据模型减少一个需同步维护的字段。
- executor 逻辑必须查询专题类型以决定执行主体。
- 变更专题类型隐式改变规划行为。
- 影响：02 tasks executor 字段保留但规划态不填；05 pm-daily 增加推断规则；01/06 Story 3 改写。
